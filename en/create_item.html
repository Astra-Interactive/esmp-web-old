EMPIRE PROJEKT Item Information:

  • Item must contain:
    Name, description (optional), texture or json model and the usual object for which it is superimposed, for example STICK
  • The item may contain craft.
  • You can set your durability.
  • Item may containattributes, which are valid when equipped in slot. You must specify how much to increase the attribute.
  • There is support for enchantments, you must write a game name
  • Interaction types:
    After eating, after drinking, on the left / right mouse button, after a monster attack, after clicking on a block, after landing (for example, a snowball or an ender pearl)
  • Events after interactions:
    Creating particle, play sound(you can add your own), command execution (from player or console), add / removepotion effect
