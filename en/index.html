  • Brief information:
There are custom items on the server, including records with music. This requires our Resource-Pack
If you want the server to have added any item, then write about it inDiscord
There is a system of ranks. You get them in /ranks after certain achievements on the server.
Roles available Guardian Oracle и Inquisition. The latter is unlikely to be received, however the first two are quite possible if you are at least Citizen.
There is a system of laws, for violation of which an appropriate punishment is imposed.
If you are at least Citizen then you can upload your own pictures via ImageMap. More in commands.
If you are at least Citizen then you can create your own clans.
If you are at least Citizen then you can create your regions / rg claim.
Regions will only save you from dynamite explosions, etc. All players will be able to break and place blocks in your region.
The server has an in-game store where you can buy any game item for the in-game currency.
